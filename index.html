Tutorial de Git e GitHub

Aprenda como usar o Git e o GitHub para controlar versões do seu código.

Introdução ao Git

Git é um sistema de controle de versão distribuído que ajuda a gerenciar mudanças no código-fonte ao longo do tempo. Ele é amplamente utilizado por desenvolvedores em todo o mundo.

Para mais informações sobre o Git, visite o site oficial do Git.

Comandos Básicos do Git

Comando Descrição
git init Inicializa um novo repositório Git.
git clone Clona um repositório existente.
git branch Gerencia branches no repositório.
git checkout Alterna entre branches ou restaura arquivos de árvore de trabalho.
git status Exibe o status do repositório.
git diff Mostra as diferenças entre commits, branches, arquivos, etc.
git add Adiciona arquivos ao índice (staging area).
git commit Grava mudanças no repositório.
git push Envia mudanças para o repositório remoto.
git pull Atualiza o repositório local com mudanças do remoto.
git revert Reverte um commit específico.
git merge Mescla branches.
git stash Salva modificações temporariamente.
git rm Remove arquivos do índice e do diretório de trabalho.
git config Define configurações do usuário.
git reset Reseta o índice e o diretório de trabalho para o estado de um commit específico.
git remote Gerencia repositórios remotos.
git fetch Baixa objetos e referências de um repositório remoto.
git show Mostra vários tipos de objetos.
git help Exibe ajuda para comandos Git.
git rebase Reaplica commits em cima de outro commit.
git blame Mostra o último commit que modificou cada linha de um arquivo.
git tag Cria, lista, deleta ou verifica tags no repositório.

Configuração e Acesso ao GitHub

O GitHub é uma plataforma de hospedagem de código que utiliza o Git para controle de versão. Para começar a usar o GitHub:

  1. Crie uma conta no GitHub.
  2. Crie um novo repositório.
  3. Conecte seu repositório local ao GitHub com o comando: git remote add origin https://github.com/seu-usuario/seu-repositorio.git
  4. Envie suas mudanças para o repositório remoto com: git push -u origin master
